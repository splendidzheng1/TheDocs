# TheDocs

这是一个基于 Jekyll 的文档网站仓库，用于发布和管理博客文章及文档内容。

## 功能特点

- 使用 Jekyll 静态网站生成器
- 支持 Markdown 格式编写文章
- 包含博客文章发布功能
- 简洁的网站结构

## 目录结构

- `_posts/`: 存放博客文章
- `_config.yml`: Jekyll 配置文件
- `about.markdown`: 关于页面
- `index.markdown`: 首页内容

## 如何使用

1. 克隆仓库
2. 安装 Jekyll 及依赖项 (`bundle install`)
3. 本地运行 (`bundle exec jekyll serve`)
4. 在 `_posts` 目录下添加新的 Markdown 文章

## 注意事项

- 文章命名格式为：`YYYY-MM-DD-title.markdown`
- 文章需包含 YAML 头信息
